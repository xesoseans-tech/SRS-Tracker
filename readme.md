# SRS//TRACKER

[![Live Demo](https://img.shields.io/badge/Live_Demo-Website-00E5FF?style=for-the-badge)](https://srstracker.netlify.app/)

**SRS//TRACKER** is a web application designed to help you lock knowledge into your long-term memory using a Spaced Repetition System (SRS). Built with a sleek, terminal-inspired dark mode UI, it automates your review schedule so you only study what you need to, exactly when you need to.

### 🚀 Live Demo
**[Visit SRS//TRACKER](https://srstracker.netlify.app/)**

---

### ✨ Features

* **Memory Protocol:** Automatically schedules reviews at scientifically proven intervals (1d, 3d, 1w, 2w, 1mo, 2mo, 4mo).
* **Cyber Aesthetic:** A clean, terminal-style interface designed for focus and minimal eye strain.
* **Secure Authentication:** Quick and secure login using Google Authentication.
* **Private Data:** Your learning data is synced securely to your account and remains 100% private.
* **Progress Dashboard:** Quickly view topics that are "Due Today," your "Total Topics," and what you have "Mastered."

### 🧠 How It Works

1.  **Add a Topic:** Enter a new subject you are learning (e.g., networking concepts, vocabulary, command line tools).
2.  **Follow the Schedule:** The tracker will notify you when a topic is due for review based on the 7-step spaced repetition timeline.
3.  **Master It:** Complete all intervals to fully commit the topic to your long-term memory and move it to the "Mastered" list.
